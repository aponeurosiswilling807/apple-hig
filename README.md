# 🍎 apple-hig - Create interfaces that match Apple standards

## 🚀 Setting up the plugin

The apple-hig plugin functions alongside Claude Code. After you install the application, integrate it with your workspace:

1. Open Claude Code on your computer.

## 🛠 Features

This tool offers several ways to improve your design:

*   **Automatic Reviews:** The plugin scans your interface files for common design errors.
*   **Accessibility Checks:** It ensures your text size and color contrast match Apple requirements for impaired users.
*   **Platform Specific Modes:** You can toggle between modes for visionOS, watchOS, or macOS.
*   **Design Token Support:** It recognizes your design tokens and suggests improvements based on Apple's standards.
*   **SwiftUI Validation:** The system reads your SwiftUI code and points out areas that break the standard look of an Apple app.

## 💡 Using the design review tool

Once you set up the tool, you can start a review. Run the review command from your terminal. The tool analyzes your code and lists issues in your terminal window. ## 🔌 Compatibility with Apple platforms

The tool contains logic for the following Apple operating systems:

*   **iOS:** Focuses on touch targets and screen layout for iPhones.
*   **iPadOS:** Covers split-screen views and multitasking design rules.
*   **macOS:** Checks for proper window behavior, menus, and keyboard control.
*   **watchOS:** Validates small-screen layout and glanceable information.
*   **tvOS:** Ensures focus states and remote control navigation work as expected.
*   **visionOS:** Reviews spatial layout and depth settings for your 3D interfaces.

## 🧩 Understanding Human Interface Guidelines

Apple provides the Human Interface Guidelines to keep all apps consistent. A consistent app is easier to learn. Users instinctively know how to interact with your software because it follows the rules they already see in every other Apple app. This tool acts as a bridge between your code and those complex design documents. It turns written rules into clear, actionable, technical feedback.
